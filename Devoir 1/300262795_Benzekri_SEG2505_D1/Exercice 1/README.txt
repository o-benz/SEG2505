Avantages et inconvénients de chaque design:
   
Design 1:
AVANTAGES:
   L'efficacité est excellente
   Le code est simple mais volumineux car il doit intégrer à la fois les modules des systèmes
DÉSAVANTAGES:
   La consommation de mémoire

Design2 : stockage des coordonnées polaires :
AVANTAGES:
   Les coordonnées polaires sont utiles pour étudier les mouvements du corps humain. C'est parce que le corps humain utilise des mouvements articulaires pivots.
   Lors de la discussion des orbites, où la symétrie radiale est impliquée, ou il y a une source ponctuelle, comme la génération d'ondulations dans l'eau, les coordonnées polaires simplifient les choses.
   Il est utilisé dans le positionnement de l'objet. Il peut être utilisé pour faire orbiter un objet autour d'un autre objet.
   La mise en œuvre est simple
DÉSAVANTAGES:
   Problèmes de balance de compteur
   Évitement de collision limité
   L'erreur de position
   Pas très efficace

Design3 : stockez uniquement les cordiates cartisiens:
AVANTAGES:
   Les calculs de distances entre points sont triviaux.
   Les calculs de superficies sont relativement faciles.
   Les représentations graphiques sont réalistes, à condition que la zone couverte ne soit pas trop grande.
   La mise en œuvre est facile
   L'efficacité est bonne par rapport à la design 2
DÉSAVANTAGES:
   Pas adaptés pour résoudre des problèmes complexes
   La consommation de mémoire augmente par rapport au design 2
 
Design 4:
AVANTAGES:
   L'efficacité est très élevée
   Presque tous les problèmes simples et complexes sont résolus par ce type de design.
   La mise en œuvre est facile
DÉSAVANTAGES:
   La complexité du code augmente
   La consommation de mémoire est la plus élevée par rapport au design 1,2,3,5 en raison de l'incorporation d'autres variables
   La simplicité du code diminue

Design 5:
AVANTAGES:
   Il est simple à coder car les deux modules sont incorporés dans le programme.
   L'efficacité du code dépend du type de problème
   La détection des erreurs est facile
DÉSAVANTAGES:
   La consommation de mémoire est supérieure à celle du design 1,2,3 mais inférieure au 4
